# simple-cataract-simulator-py

A simple **text-only cataract simulator** written in Python.  
This project is not realistic for actual vision . Designed as a learning project to understand text manipulation and basic simulation effects in Python.

Working entirely in the console. You can enter any text to simulate "cataracts". Adjust severity from 1 to 5(mild to string)

## How to Run

1. Make sure you have **Python 3** installed
2. Download or clone this repository

## Running the Executable

If you don’t want to run the Python script, you can also use the **pre-built Windows executable** included in this repository.  

1. Download the `.exe` from the `dist` folder in this repo.
2. Double-click the file `main.exe`

NOTE: This is text-only, so it cannot truly represent cataract vision!!
